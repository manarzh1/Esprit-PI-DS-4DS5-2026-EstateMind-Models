# Estate Mind – AI & Data Science

## Overview

This repository contains artificial intelligence models developed for the Estate Mind project.

It includes Machine Learning, Deep Learning, and Explainable AI techniques to support real estate decision-making.

## Features

* Data analysis and preprocessing
* Machine Learning models (XGBoost, LightGBM)
* Deep Learning models (Neural Networks)
* Price prediction
* Risk detection
* Explainability (SHAP, XAI)

## Tech Stack

* Python
* Scikit-learn
* XGBoost / LightGBM
* TensorFlow / PyTorch
* SHAP

## Structure

* models
* notebooks
* data
* utils
